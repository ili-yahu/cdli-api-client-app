Help

            
Use the CDLI API Client by entering commands in the input box.

            
Commands:

            
    
                
  • export: Export catalog and text data
    • 
                
  • search: Search the catalog
    • 
            

            
Options:

            
    
                
  • --host: Specify the host URL
    • 
                
  • --format: Specify the output format (ndjson, csv, etc.)
    • 
                
  • --output-file: Specify an output file
